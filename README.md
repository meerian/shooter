## Features
Current features include:
* Scoring system
* Pause feature
* Experience and leveling up

Coming soon:
* Varied enemies
* Leveling up system
* More weapon types

## How to play:
1. Press the `start` button to start the game
1. Use arrow keys to control your character


## Tech Stack:
Made using Javascript with PixiJS API and Tailwind CSS
